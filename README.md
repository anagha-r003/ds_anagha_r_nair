# Trader Behavior Analysis under Fear and Greed

This project analyzes how trader behavior changes under different market
sentiment conditions such as Fear and Greed using historical trading data
and a Bitcoin market sentiment index.

## Contents
- notebook_1.ipynb : Google Colab notebook with full analysis
- csv_files/       : Processed and intermediate CSV data
- outputs/         : Graphs and visualizations
- ds_report.pdf    : Final summarized report
